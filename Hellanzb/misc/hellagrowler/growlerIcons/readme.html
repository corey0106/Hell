From the Artist

August 17th, 2004
Jolly Roger by Anthony Piraino


This icon set puts you on the high seas, seeking fame (or infamy!), fortune and glory as a pirate. I've provided a few pirate friends, a flag, and some treasure to get you started. The rest is up to you!

My inspiration for this set came from a souvenir my sister bought for my son during a vacation in Florida -- a pirate head carved out of a coconut. The Red Jim Rackam icon is based on the coconut pirate head.
